# Фабрика Каркасов Алатау — статический сайт

Многостраничный сайт без React и сборщиков. Весь контент доступен в чистом HTML5, стили — в одном CSS-файле. JavaScript необязателен и используется только для улучшения UX (лайтбокс, закрытие мобильного меню).

## Структура проекта
```
frontend/
├── assets/
│   ├── css/styles.css      # Общие стили и сетка
│   ├── js/enhancements.js  # Необязательные улучшения UX
│   └── img/                # Изображения (герои и галерея)
├── index.html              # Главная
├── capsules.html           # Капсулы
├── hangars.html            # Ангары
├── garages.html            # Гаражи
├── documents.html          # Документы
├── o-nas.html              # О компании
└── contacts.html           # Контакты и форма обратной связи
```

## Progressive enhancement
- **Меню**: CSS-only на checkbox. При выключенном JS раскрывается и закрывается стандартно; скрипт только закрывает меню после клика.
- **Галерея**: работает через `:target`, поэтому открывается без JS; скрипт добавляет плавный оверлей.
- **Формы**: отправляются напрямую (Formspree). Без JS доступны все поля и подсказки.

## Запуск
Откройте любой HTML-файл напрямую в браузере или поднимите простой сервер:
```
python3 -m http.server 8080 --directory frontend
```
