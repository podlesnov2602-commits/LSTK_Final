# Статический сайт «Фабрика Каркасов Алатау»

Многостраничный сайт без сборщиков и зависимостей. Вся информация доступна в HTML, JavaScript используется только для улучшения UX (закрытие мобильного меню, мягкий лайтбокс).

## Структура
```
frontend/
├── assets/
│   ├── css/styles.css      # Общие стили
│   ├── js/enhancements.js  # Необязательные улучшения UX
│   └── img/                # Статические изображения
├── index.html              # Главная
├── capsules.html           # Капсулы
├── hangars.html            # Ангары
├── garages.html            # Гаражи
├── documents.html          # Документы
├── o-nas.html              # О компании
└── contacts.html           # Контакты
```

## Progressive enhancement
- Меню работает через checkbox (CSS-only); JS только закрывает его после клика.
- Галерея открывается через `:target`; JS добавляет плавный оверлей.
- Формы отправляются напрямую (Formspree), без зависимости от JS.

## Просмотр
Откройте любой HTML-файл напрямую в браузере или раздайте папку `frontend` через простой веб-сервер:
```
python3 -m http.server 8080 --directory frontend
```
